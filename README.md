# bustaTv

Una plataforma web de streaming de contenido similar a Míralo TV.

## 📋 Descripción

bustaTv es un proyecto de agregador de streams de contenido en vivo. La arquitectura está basada en el análisis de miralotv.com.

## 🏗️ Arquitectura

```
bustaTv/
├── frontend/           # Frontend (HTML, CSS, JavaScript)
├── backend/            # Backend (API, gestión de streams)
├── docs/              # Documentación
└── scripts/           # Utilidades y scripts
```

## 🛠️ Tecnologías

### Frontend
- HTML5
- CSS3
- JavaScript vanilla / Vue.js / React
- **Clappr** - Reproductor de video HTML5

### Backend
- Node.js / Python / Go
- API REST

### Herramientas de Scraping
- **Firecrawl** - Extracción de contenido web
- **Playwright** - Automatización de navegador

## 📦 Instalación

```bash
git clone https://github.com/jobustamantedev/bustaTv.git
cd bustaTv
```
